# Инструкция для работы с Markdown

## Выделение текста

Чтобы виделить текст курсивом необходимо его обрамить звездочкамми (*), Например, *вот так*.

Чтобы выделить текст полужирным, необходимо обрамить его двойнымми звездочками (**).
Например, **вот так**.
